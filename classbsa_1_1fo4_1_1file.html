Parameters

a_format The format to read the file as.
a_mipChunkWidth The maxiumum width to restrict a single mip chunk to.

a_level The level to compress the data at.
a_compression The resulting compression of the file read.
Exceptions
binary_io::buffer_exhausted Thrown when reads index out of bounds.
std::system_error Thrown when filesystem errors are encountered.

Remark

If any exception is thrown, the object is left in an unspecified state. Use clear to return it to a valid state.

If the function returns successfully, the contents of the object are replaced with the contents of the input.

Remark

If std::system_error is thrown, the file is left unmodified.
