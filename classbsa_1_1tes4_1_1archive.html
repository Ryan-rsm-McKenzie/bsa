auto bsa::tes4::archive::read(std::filesystem::path a_path) -> version

Reads the contents of the archive from disk.

Parameters
a_path The path to the given archive on the native filesystem.
Returns The version of the archive that was read.
Exceptions
std::system_error Thrown when filesystem errors are encountered.
bsa::exception Thrown when archive parsing errors are encountered.

Remark

If std::system_error is thrown, the archive is left unmodified.

If bsa::exception is throw, the archive is left in an unspecified state. Use clear to return it to a valid state.

If the function returns successfully, the contents of the archived are replaced with the contents of the archive on disk.
